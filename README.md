# ArcGlobal

A lightweight web page that showcases the **ArcGlobal** branding with a modern, responsive design.

## Features
- Custom logo (logo.png) displayed in header and footer
- Fixed header with smooth scrolling navigation
- Hero section with a call‑to‑action button
- Responsive layout that works on desktop and mobile devices
- Clean, modern styling using CSS variables and Flexbox

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/nishadnoureen-debug/arcmanpower.git
   cd arcmanpower
   ```
2. Open `index.html` in any web browser to view the site.

## Project Structure
- `index.html` – Main HTML page
- `styles.css` – All styling for the site
- `script.js` – Small JavaScript for interactive elements (e.g., navigation toggle)
- `logo.png` – Brand logo image
- `README.md` – This documentation file

## Development
Feel free to customize the colors, text, or layout. The CSS variables at the top of `styles.css` make theming easy.
